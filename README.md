# 简介
一个游戏引擎开发计划    
底层使用C++ 
上层使用beef-lang   
目标是 面向数据设计，ECS，高性能，高灵活性和可扩展性    
目前在Windows-DX12下进行开发，Windows平台开发完毕后会进行其它平台的开发
